LOAN APPROVAL WORKFLOW:
======================

The system uses a maker-checker pattern for loan approval:

1. MEMBER applies for a loan (via loan application form)
2. TREASURER reviews and approves the loan
3. CHAMA_ADMIN gives final approval
4. For disbursement:
   - TREASURER approves the disbursement first
   - CHAMA_ADMIN then approves the disbursement
   - Only then is money sent via M-Pesa
